Key files are an optional authentication method. More info on key files is available on the KeePass site

CKP can store your key files locally in Chrome storage, and apply them when opening your password database.

Websites and other Chrome extensions do not have access to these files. However, they are stored unencrypted in your local Chrome profile and someone with access to your device could read them.
