CKP is an extension that uses your existing KeePass database files to autofill passwords on websites.

The extension is only visible when you go to a web page that has input fields. When it activates, it shows a small KeePass icon on the right hand side of the browser address bar. Clicking on the icon will give you options to connect to your KeePass database, unlock it, and then autofill in the password on the website.

To continue, you should...
  1. Click on the "KeePass Databases" menu to show it where to find a KeePass database.
  2. Go to a website with a login page, and try it out. (for example)
If your database requires a key file then you should also click on the "Key Files" menu to load your key file.
